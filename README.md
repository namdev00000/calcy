## Keyboard Shortcuts

*   **Numbers (0-9) and Decimal (.):** Input corresponding digits.
*   **Operators (+, -, \*, /):** Perform arithmetic operations.
*   **Enter or =:** Calculate the result.
*   **Backspace or Delete:** Delete the last input.
*   **Escape:** Clear all (`AC`).
*   **Inside Feature Popups:**
    *   **Enter:** Execute the feature calculation.
    *   **Escape:** Clear input or close the popup if inputs are empty.
